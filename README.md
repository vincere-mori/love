# Пион

Генеративный 3D-пион на p5.js. Растёт из бутона, крутится, увядает, потом всё повторяется со следующим цветком — и так по кругу.

Демо: https://vincere-mori.github.io/pion-p5/

## Что внутри

- Три варианта цветка с разными палитрами.
- Три режима отрисовки переключаются по очереди: ASCII (кириллица), точки, пиксели.
- Авто-вращение по трём осям + лёгкий доворот за курсором.
- Глитч с цветовым сдвигом — срабатывает при смене режима, увядании и иногда сам.

## Управление

- мышь — наклон камеры
- клик / `G` — глитч вручную
- `F` — полноэкранный режим

## Запуск

Один файл, без сборки. Открой `index.html` в браузере либо подними любой статический сервер:

```bash
python -m http.server 8000
```
